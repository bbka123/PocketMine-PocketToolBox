#### 功能：
1. 自动清理
2. 自动备份
3. 超开检测
4. 性能评测
5. 实时数据广播
6. 自动移除出错插件
7. 崩服邮件提示
***
#### 指令集
```bash
ptb test        #开启跑分测试，测试cpu、io、同服mc服务器数量并加入全国排行
ptb status      #查看服务器实时信息
ptb ischaokai   #判断是否超开
ptb backup      #服务器备份
```
