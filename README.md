# 口袋工具箱（PocketToolBox）
**PocketMine工具箱、跑分测试**

#### 功能：
目前支持跑分测试、实时信息广播、文件清理
- 自动清理
- 自动备份
- 超开检测
- 性能评测
- 实时数据广播
- 自动移除出错插件
- 崩服邮件提示

#### 指令集
```bash
ptb test        #开启跑分测试，测试cpu、io、同服mc服务器数量并加入全国排行
ptb status      #查看服务器实时信息
ptb ischaokai   #判断是否超开
ptb backup      #服务器备份
```
